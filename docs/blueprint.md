# **App Name**: ClientVerse

## Core Features:

- Add Customer Profile: Create a new customer profile with fields for customer name, phone model, phone IMEI, phone price, and transaction date.
- Edit Customer Profile: Edit existing customer profiles to update information.
- Image Upload: Upload one or more images to each customer profile to capture things like the phone's condition at the time of sale.
- Generate Invoice: Generate a customer invoice from the profile information using an AI tool. This feature intelligently incorporates customer details and relevant product information to create a professional invoice, while taking into account that this invoice tool is for Chinese users.
- Download Invoice: Preview and download generated invoices in a standard format such as PDF.
- Search Profiles: Search functionality to quickly find customer profiles.

## Style Guidelines:

- Primary color: Light blue (#7EC4CF) to evoke a sense of trust and reliability.
- Background color: Very light blue (#E0F7FA), near-white.
- Accent color: Orange (#F2943A) for highlighting calls to action and important information.
- Body and headline font: 'PT Sans' (sans-serif) for a clean and modern user interface.
- Simple, outline-style icons for clarity and ease of understanding.
- Clean and organized layout with a clear visual hierarchy to ensure ease of use.
